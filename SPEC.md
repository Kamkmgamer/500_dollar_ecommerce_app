# Boutique Store - $500 Budget

## Overview
- **Budget**: $500
- **Target Business**: Local business / Boutique
- **Max Products**: 200-1000

## Tech Stack
- Next.js + Shopify Lite API
- Simulated with Next.js for demo purposes
- Artificial 200ms delay on all pages to simulate slower performance
- Basic product management

## Features
- Product management
- Shopping cart
- Basic checkout
- Order management
- Simple analytics
- Intentionally slowed performance

## Implementation
- Next.js with artificial delay middleware
- Simulates slower PHP/Shopify performance
- No advanced optimization
- Basic caching only

## Performance (Intentionally Slower)
- Artificial 200ms delay on all pages
- Demonstrates performance difference from higher tiers
- Expected: 1-1.5s page load time (with delay)
